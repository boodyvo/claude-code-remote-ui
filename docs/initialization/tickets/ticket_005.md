## Implementation Notes

### Dockerfile

```dockerfile
FROM node:22-alpine AS deps
WORKDIR /app
RUN corepack enable
COPY package.json pnpm-lock.yaml ./
RUN pnpm install --frozen-lockfile

FROM node:22-alpine AS builder
WORKDIR /app
RUN corepack enable
COPY --from=deps /app/node_modules ./node_modules
COPY . .
RUN pnpm build

FROM node:22-alpine AS runner
WORKDIR /app
ENV NODE_ENV=production

# Install Claude Code CLI
RUN npm install -g @anthropic-ai/claude-code

# Non-root user
RUN addgroup --system --gid 1001 app && \
    adduser --system --uid 1001 --ingroup app app

# Copy standalone build
COPY --from=builder /app/public ./public
COPY --from=builder --chown=app:app /app/.next/standalone ./
COPY --from=builder --chown=app:app /app/.next/static ./.next/static

# Data directories
RUN mkdir -p /home/app/.claude /app/workspace /app/data && \
    chown -R app:app /home/app/.claude /app/workspace /app/data

USER app
EXPOSE 3000
ENV PORT=3000 HOSTNAME=0.0.0.0
CMD ["node", "server.js"]
```

### docker-compose.yml

Per product design §9.2 with Coolify labels, 3 named volumes, health check.

### Coolify-Specific Notes

- **Disable Gzip** in Coolify advanced settings (breaks WebSocket/SSE through Traefik)
- Use **Raw Compose Deployment** mode
- Set env vars in Coolify UI: `DEEPGRAM_API_KEY`, `SESSION_SECRET` (use `SERVICE_PASSWORD_64_SESSION`)
- `SERVICE_FQDN_CONNECTOR` auto-generated by Coolify

### Volumes

| Volume | Mount | Purpose |
|--------|-------|---------|
| `claude_home` | `/home/app/.claude` | Auth tokens, session JSONL, config |
| `workspace` | `/app/workspace` | Project files |
| `app_data` | `/app/data` | SQLite DB, app settings |

### First-Run on Hetzner

1. Deploy via Coolify
2. Open terminal in Coolify dashboard for the container
3. Run `claude login` to authenticate with Max subscription
4. Token is saved to persistent volume — survives redeploys
